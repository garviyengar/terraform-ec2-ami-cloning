# EC2 Provisioning and AMI Cloning with Terraform

Infrastructure as Code project demonstrating automated EC2 deployment, custom AMI creation and instance cloning using Terraform.

## Overview

This project provisions an EC2 instance, creates a custom Amazon Machine Image (AMI), and launches a cloned EC2 instance from that AMI.

## Architecture

```text
Terraform
    │
    ▼
EC2 Instance
    │
    ▼
Custom AMI
    │
    ▼
Cloned EC2 Instance
```
